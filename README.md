# 📈 Previsão do Preço da Soja com Machine Learning

Projeto de **Machine Learning aplicado ao Agronegócio** que prevê o preço da soja com base em dados históricos.

O sistema utiliza algoritmos de aprendizado de máquina para analisar dados históricos e gerar previsões futuras.  
Também possui um **dashboard interativo desenvolvido com Streamlit** para facilitar a visualização e utilização do modelo.

---

# 🌱 Motivação

O agronegócio é um dos setores mais importantes da economia brasileira.  
Prever o comportamento de commodities agrícolas pode ajudar em:

- tomada de decisão
- planejamento de produção
- análise de mercado

Este projeto foi desenvolvido como parte dos meus estudos em **Inteligência Artificial e Machine Learning**, com foco em aplicações no **AgTech (Agricultural Technology)**.

# 📂 Estrutura do Projeto
